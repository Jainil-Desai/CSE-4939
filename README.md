1) git clone this repo
2) cd into STAC-IT
3) run `npm install`
4) run `npm install axios`
5) run `npx expo start`
6) choose between w, a, or i.
   - w for web
   - a for andriod studio
   - i for iOS simulator

to run backend:
1) open a new terminal window
2) cd into backend directory
3) create a venv and activate it
4) run `pip install huggingface_hub django requests djangorestframework`
5) run `pip install django-cors-headers`
6) to start the server run `python manage.py runserver`
